### Setup

1. Installation
```
git clone git@github.com:symbo.ls/uikit.git
```

2. Import the component from Symbols
```
import { Box } from 'smbls'
```

3. Use it inside your DOMQL code
```
const Header = {
  proto: Box,
  // ...Other Properties
}
```
