# DOMQL Starter Kit
Example dev setup (boilerplate) to use [DOMQL](https://github.com/rackai/domql). You can also check [playground](https://domql.com/playground/).

### Setup

1. Clone the repo
```
git clone git@github.com:domql/starter-kit.git
```

2. Install scripts
```
yarn
```

3. Run the project
```
yarn start
```
